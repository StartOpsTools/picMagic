# picMagic

picMagic 是一个简单的图片处理的web工具

## feature

1. 提供原图大小剪切功能

## 约定

1. 当不使用样式访问的时候 (即: 无样式)，直接返回原图
2. 当样式不匹配的时候 (即: 无样不存在样式表中)，直接返回原图

为什么使用!进行样式判断:
    使用!进行样式判断 -- 更好的命中缓存 (!是uri中的一部分)

filepath中需要杜绝 ! 号的出现

存在以下问题:
1.  当路径中出现!，但是没有给出样式 (路径为 /img/avatar/q!x.png)
    这个时候就会把:
        x.png 当成样式拦截
        /img/avatar/q 当成路径
        /img/avatar/q 为一个无格式文件

## 图片处理

图片效果:
    亮度
    对比度
    锐化
    模糊

缩略效果:
    高度
    宽度

水印设置:
    文字水印
    图片水印
